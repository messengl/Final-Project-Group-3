Data Shredder

A website for easy data organization.
Our website is dedicated to taking data sets and organizing them into easily readable graphs and tables. You can select a data set and choose if you want to see it in a pie chart, a plot, a bar graph.

Additionally, you can upload your own data set and select which type of graph you would like to view it in. This feature is coming soon.
