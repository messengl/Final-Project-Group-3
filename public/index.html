A website for simple data organization.

Our website is dedicated to taking data sets and organizing them into easily readable graphs and tables. You can select one of our data sets and choose if you want to see it in a pie chart, a plot, bar graph, or a table.

Additionally, you can upload your own data set and select which type of graph you would like to view it in.
